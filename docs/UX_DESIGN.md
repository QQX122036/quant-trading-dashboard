# UX Design - VeighNa Web Dashboard

> 版本: v4.0 | 更新日期: 2026-04-05

---

## 1. 设计理念

### 1.1 设计语言
- **风格**: Fintech Dark + Glassmorphism + Bento Grid
- **对标**: Bloomberg Terminal / TradingView 专业级水准
- **特点**: 信息密度高、实时响应快、金融数据可视化精准

### 1.2 色彩系统 (A股红涨绿跌)

```
背景色系:
- 页面背景: #0A0E17 (深空黑)
- 卡片背景: #111827 (暗灰)
- 玻璃背景: rgba(17, 24, 39, 0.8) + backdrop-blur
- 边框色: rgba(255, 255, 255, 0.1)

主题色 (A股标准):
- 上涨/多头: #EF4444 (红色)
- 下跌/空头: #22C55E (绿色)
- 强调色: #3B82F6 (科技蓝)
- 警告色: #F59E0B (琥珀黄)

文字色:
- 主文字: #F9FAFB
- 次文字: #9CA3AF
- 弱文字: #6B7280
```

## 5. 技术实现

### 5.1 技术栈

| 层级 | 技术 | 说明 |
|-----|------|------|
| **框架** | SolidJS 1.8 + TypeScript | 响应式前端 |
| **构建** | Vite 6 | 极速 HMR |
| **样式** | TailwindCSS v4 | 原子化 CSS |
| **图表** | lightweight-charts v4 (K线/指标) + ECharts v5 (收益率) | 专业金融图表 |
| **状态** | SolidJS Signal/Store | 内置响应式 |
| **路由** | @solidjs/router | 页面路由 |

### 5.2 路由结构

```typescript
/                    → /market (市场总览)
/market              → 市场总览 (默认页)
/dashboard           → A股看板 (K线图)
/backtest           → 回测分析
/trades              → 交易记录
/positions           → 持仓管理
/data                → 数据采集
```

## 7. 下一步优化建议

### 7.1 高优先级
1. **WebSocket 实时数据对接** - 连接后端获取真实行情
2. **因子信号标记** - 在K线图上显示买入/卖出信号
3. **自选股管理** - 添加/删除自选股

### 7.2 中优先级
4. **回测历史记录** - 查看历史回测结果
5. **数据导出功能** - 导出CSV/Excel
6. **快捷键支持** - Esc关闭弹窗, Enter确认

### 7.3 低优先级
7. **主题切换** - 暗色/亮色模式
8. **国际化** - 中文/英文切换
9. **移动端适配** - 响应式布局
